# 🌦️ Weather App

A simple and responsive **Weather Application** that shows real-time weather data using an API. Users can search any city and get current weather details instantly.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* 💨 Wind speed & humidity info
* 🌥️ Weather condition (Cloudy, Sunny, Rainy, etc.)
* 📱 Responsive design (Mobile + Desktop)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Weather API (e.g., OpenWeather API)
